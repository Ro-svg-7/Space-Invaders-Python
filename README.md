# 🚀 Space Invaders Clone

A Space Invaders-inspired arcade game built with Python and Turtle Graphics.

## Features

- 🎮 Player-controlled spaceship
- 🔫 Bullet shooting system
- 👾 Alien formation with multiple rows
- ❤️ Multi-hit enemy system
  - Top row aliens require 1 hit
  - Middle row aliens require 2 hits
  - Bottom row aliens require 3 hits
- 🎨 Visual health indicators using color changes
- 📈 Real-time score tracking
- 🏆 Win and Game Over screens
- 🔄 Restart functionality
- ↔️ Classic side-to-side alien movement
- ⚡ Dynamic difficulty scaling as aliens are eliminated

## Controls

| Key | Action |
|-----|---------|
| ← | Move Left |
| → | Move Right |
| Space | Shoot |
| r | Restart Game |
